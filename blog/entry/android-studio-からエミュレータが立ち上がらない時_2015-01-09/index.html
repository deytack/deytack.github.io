Android Studio からエミュレータが立ち上がらない時

Created: 2015-01-09 02:07:00 +0900

Android Studio をインストールして、適当なサンプルファイルを Run しても、 エミュレータに Android のロゴが表示されるだけで、10 分くらい待てどもそこから先に進まない。

こんなエラーっぽい文言がコンソールに出ていたので調べる。

emulator: The memory needed by this VM exceeds the driver limit.
HAX is not working and emulator runs in emulation mode

どうやら HAX (= Hardware Accelerated Execution) という仕組みを有効化しないと 僕のマシンではまともに動いてくれないようだった。

ここを参考に HAX Manager をインストールする。

  • Androidエミュレータは遅い！重い！！Intel x86設定で高速化して使うべし | サクッとJava研修室

で、設定がうまくいったようでコンソールに出る文字がこんなものに変わった。

HAX is working and emulator runs in fast girt mode

結果、とりあえず２分くらい待ったら起動してくれるようになった。 に、２分…。2012 年の MacBook だとこんなもんなのか…。

まあ、これからの検証でそんなに頻繁にエミュレータを立ち上げるわけでもなさそうだしいいか…。
